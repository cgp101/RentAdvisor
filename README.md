## RentAdvisor
Telegram chatbot for US rental market insights using Zillow ZORI data, this can be done for any place as long as we the rental data for the region. 

## Tech Stack 
- n8n (workflow automation)
- Telegram Bot API
- Google Sheets (Zillow ZORI data)
- Ollama (Llama 3.1, Mistral) --> We can use a cloud provider instead of a local host.

## How to use 
Import `rent_advisor_v1.json` into n8n
2. Configure credentials (Telegram, Google Sheets, Ollama)
3. Activate workflow
